📘 PWHBeta — TOEIC Vocabulary Trainer

PWHBeta — это Progressive Web App (PWA) для изучения слов TOEIC.  
Приложение работает прямо в браузере и поддерживает офлайн‑режим благодаря Service Worker.

 🚀 Возможности
- Изучение слов TOEIC с удобным интерфейсом.
- Повторение слов и отслеживание прогресса.
- Поддержка офлайн‑режима (PWA).
- Автоматический деплой через GitHub Actions и GitHub Pages.

 🛠️ Технологии
- HTML, CSS, JavaScript
- Service Worker (офлайн‑режим)
- GitHub Actions (CI/CD)
- GitHub Pages (деплой)
